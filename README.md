# EduLink

A role-based education portal for students, lecturers, and administrators.

## Getting Started

1. Run `npm install`
2. Run `npm run dev`
3. Open the app in your browser

## Admin Login

A default admin account is already seeded into the app for local testing.

- Role: Admin
- Email: `admin@univ-iug.com`
- Password: `Admin@123`

### How to sign in as admin

1. Open the login page.
2. Select the Admin option.
3. Enter the email and password above.
4. You will be redirected to the admin dashboard.

### Admin capabilities

After logging in, the admin can:
- View the admin dashboard
- Create lecturer and student accounts
- Set or view account passwords
- Publish announcements
- Manage courses and users

> Note: Accounts are stored locally in the browser for this demo version.
